# 🏟️ Smart Stadium System

An AI-powered system to improve stadium experience by managing crowd flow, reducing waiting times, and providing smart services.

## 🚀 Features
- Live Crowd Monitoring
- Smart Area Suggestions
- Queue Time Prediction
- In-Seat Food Ordering
- Emergency Support System

## 🛠️ Tech Stack
- Python
- Streamlit

## ▶️ Run Locally
streamlit run app.py
